# Flappy Bird Codex 5.5

Static browser game built for GitHub Pages.

## What changed

- stronger Codex 5.5 front-end presentation
- near-miss streak feel and fever windows
- dynamic visual shifts during hotter runs
- zero-build static deploy from repo root

## Controls

- `Space`, `Up`, click, or tap: flap

## Deploy

Enable GitHub Pages from the `main` branch and `/ (root)`.
